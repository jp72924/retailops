✎ Development — console email backend
The email was printed to the terminal running the Django server. Look for the block that starts with Content-Type: text/plain and find the line beginning with http:// — that is your reset link. Copy the entire URL on that line (it is now on a single, unbroken line).
✉ Production — send real emails
Add the following to your .env file (or set as environment variables) to switch to a real SMTP sender: 
DJANGO_EMAIL_BACKEND=django.core.mail.backends.smtp.EmailBackend
DJANGO_EMAIL_HOST=smtp.gmail.com
DJANGO_EMAIL_PORT=587
DJANGO_EMAIL_USE_TLS=True
DJANGO_EMAIL_HOST_USER=you@gmail.com
DJANGO_EMAIL_HOST_PASSWORD=your-app-password
DJANGO_DEFAULT_FROM_EMAIL=RetailOps <you@gmail.com>
For Gmail, generate an App Password at myaccount.google.com → Security → 2-Step Verification → App passwords.

Didn't receive it? Try again or ask your system administrator to reset your password directly.
